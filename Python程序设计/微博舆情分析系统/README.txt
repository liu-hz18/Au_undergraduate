1.安装matplotlib: 命令行下输入pip install matplotlib
2.安装snownlp :命令行下输入 pip install snownlp
3.安装scipy: 命令行下输入 pip install scipy
4.安装wordcloud: 法1：命令行下输入conda install -c conda-forge wordcloud 
	            法2：（如果法1不行）压缩包下解压word_cloud-master.zip，从命令行进入解压后的目录，然后键入python setup.py install
5.打开test.py,应该可以运行该文件（其他py文件不要动！）
 good luck :)
